# Codex Pulse Demo

A cinematic single-page frontend demo designed for GitHub Pages.

## Live URL

Once GitHub Pages is enabled for the repository, the site is intended to be served from:

`https://alan7414.github.io/codex`

## Files

- `index.html`: the full interactive landing page
- `.nojekyll`: ensures GitHub Pages serves the site without Jekyll processing

## Deploy

In the GitHub repository settings:

1. Open **Pages**
2. Set **Build and deployment** to **Deploy from a branch**
3. Select branch `main`
4. Select folder `/ (root)`
5. Save

GitHub Pages will then publish the demo site.
