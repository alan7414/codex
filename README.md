## 访问方式

启用 GitHub Pages 后，演示地址为：

`https://alan7414.github.io/codex`

## 仓库文件

- `index.html`: 单页动态演示主文件
- `.nojekyll`: GitHub Pages 静态托管兼容文件
- `.github/workflows/deploy-pages.yml`: GitHub Pages 自动部署工作流

## 部署说明

在仓库 `Settings -> Pages` 中：

1. 将 `Source` 设置为 `GitHub Actions`
2. 等待 `Deploy GitHub Pages` 工作流运行完成
3. 访问 `https://alan7414.github.io/codex`
